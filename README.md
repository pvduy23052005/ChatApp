# Real-time Chat Application

> Ứng dụng nhắn tin thời gian thực hiện đại, hỗ trợ chat 1-1, chat nhóm, quản lý thành viên và chia sẻ đa phương tiện. Được xây dựng bằng **Node.js**, **TypeScript** và **MongoDB**.
